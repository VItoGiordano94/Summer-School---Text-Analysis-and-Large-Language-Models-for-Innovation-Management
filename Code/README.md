# Code Developed During the Lessons

This folder contains the **R scripts and code examples developed during the lessons** of the **Summer School on Text Analysis and Large Language Models for Innovation Management**, organised by the University of Pisa.

The code included in this folder was written, discussed, or modified during the classroom sessions to demonstrate the practical implementation of the concepts and methods presented in the slides.

## How to Use the Code

Participants should clone or download the repository and open the relevant script in **RStudio** or another R development environment.

Before running a script, participants should:

1. verify that the required datasets are available;
2. check the file paths used in the script;
3. install and load the required R packages;
4. run the code progressively rather than executing the entire script at once;
5. examine the objects, outputs, tables, and visualisations generated;
6. read the comments included in the code;
7. compare the code with the corresponding lesson slides.

The scripts are intended to support learning and experimentation. Participants are encouraged to modify the code, test different parameters, and adapt the examples to their own research questions.
## Code Status

Some files may represent code developed live during a lesson and may therefore contain:

* intermediate analytical steps;
* comments and annotations added during the class;
* alternative versions of the same analysis;
* exploratory code;
* incomplete sections;
* examples created for teaching purposes.

The code should not necessarily be interpreted as final, optimised, or ready for direct use in a research project.

Participants should carefully review, validate, and adapt the scripts before applying them to other datasets or research contexts.

## Use of the Materials

The code is intended primarily for educational and research purposes within the Summer School.

Participants may adapt the scripts to support their own learning and research activities. When reusing or adapting substantial parts of the code, please acknowledge the relevant instructor and the **Summer School on Text Analysis and Large Language Models for Innovation Management**.
