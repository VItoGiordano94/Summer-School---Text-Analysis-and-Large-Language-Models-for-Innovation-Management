# Exercises

This folder contains the **R scripts, datasets, instructions, and supporting files** used during the practical exercises of the **Summer School on Text Analysis and Large Language Models for Innovation Management**, organised by the University of Pisa.

The exercises are designed to help participants apply the concepts and methods introduced during the lessons. They provide hands-on experience with the main stages of a text-analysis project, from importing and preparing textual data to applying analytical methods and interpreting the results.

## How to Use the Exercises

Participants should download or clone the repository and open the relevant R script in **RStudio** or another R development environment.

Before running an exercise, participants should:

1. read the instructions included in the script or in the exercise-specific README file;
2. verify that all required datasets are available;
3. install the necessary R packages;
4. set the correct working directory or use an R project;
5. run the code progressively and examine the resulting outputs;
6. complete the requested tasks and interpret the results.

Participants are encouraged to modify the code, test alternative parameters, and adapt the exercises to their own research questions and datasets.

## Use of the Materials

These exercises are intended primarily for participants in the Summer School and may be used for educational and research purposes.
